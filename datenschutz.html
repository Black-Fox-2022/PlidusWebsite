Diese Website wird über GitHub Pages (GitHub Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA) gehostet. Beim Aufruf der Website werden automatisch durch GitHub technische Verbindungsdaten erhoben und in Server-Logfiles gespeichert:

  • IP-Adresse des anfragenden Rechners

Die App Plidus wird über den Apple App Store vertrieben. Beim Herunterladen der App verarbeitet Apple personenbezogene Daten entsprechend der eigenen Datenschutzrichtlinie. 6. Datenverarbeitung in der App

Plidus wurde bewusst datensparsam gestaltet. Wir betreiben keine eigenen Server und speichern keine Nutzerdaten.

Die App verarbeitet folgende Daten:

  • Vorname: Wird ausschließlich lokal auf deinem Gerät gespeichert, um dich persönlich anzusprechen. Wird nicht an Server übertragen.
  • Antworten auf die Fragen: Werden lokal auf deinem Gerät gespeichert. Zur Erstellung deiner Studienempfehlung werden sie einmalig an die KI-API gesendet (siehe Abschnitt 7). Sie werden nicht auf unseren Servern gespeichert, da wir keine eigenen Server betreiben.
  • Favoriten und Einstellungen: Werden ausschließlich lokal auf deinem Gerät gespeichert.
  • Studiengangsdaten: Stammen von der Bundesagentur für Arbeit und sind öffentlich zugänglich. Diese Daten sind nicht personenbezogen.

7. Nutzung von KI (Claude / Anthropic)

Zur Analyse deines Profils und zur Beantwortung von Fragen zu Studiengängen nutzt Plidus die API von Anthropic PBC (548 Market St, PMB 90375, San Francisco, CA 94104, USA). Dabei werden deine Fragebogen-Antworten zur Verarbeitung an Anthropic gesendet.

Wichtige Hinweise zur Datenverarbeitung durch Anthropic:

  • Anthropic speichert bei API-Nutzung standardmäßig keine Eingaben und Ausgaben zur Modellverbesserung.
  • Die Daten werden nur zur Verarbeitung der aktuellen Anfrage verwendet.
  • Es werden keine Benutzerkonten oder Profile bei Anthropic erstellt.

Die Rechtsgrundlage ist Art. 6 Abs. 1 lit. b DSGVO (Vertragserfüllung — die KI-Analyse ist Kernfunktion der App). Weitere Informationen: Datenschutzerklärung von Anthropic.
